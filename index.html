Rules
  1. Ships can be placed horizontally or vertically, but not diagonally.
  2. All ships must be placed.
    • 1 x 5 length ship
    • 2 x 4 length ship
    • 3 x 3 length ship
    • 4 x 2 length ship
  3. Ships cannot overlap each other.
  4. Ships cannot touch each other.
